# Kali-Updater


### INSTALL

- Clone Repository

	```
	git clone https://github.com/Cashiuus/Kali-Updater.git
	```

- Modify kaliUpdater.py script with desired configuration settings

- Run script (no arguments necessary at this time)
	```python kaliUpdater.py```

### SUPPORT

Tested on versions:
- 2.7.6
- 3.3.5
